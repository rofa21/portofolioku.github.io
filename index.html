My Projects

Proyek yang pernah saya buat adalah sebuah aplikasi website admin untuk toko jam dengan fitur CRUD sederhana. Website ini memiliki tiga menu utama: Produk, Pembelian, dan Pelanggan. Selain itu, terdapat menu tambahan seperti Dashboard dan Logout untuk memudahkan navigasi pengguna. Anda dapat mengunjungi website ini melalui tautan berikut: http://rofapro.free.nf/login.php.

Pada website admin ini, selain fitur CRUD, saya juga menambahkan Filter Produk Untuk mempermudah pencarian produk berdasarkan kriteria tertentu. Detail Pembelian Menampilkan informasi lengkap setiap transaksi.Autentikasi Admin harus login untuk mengakses dan logout untuk keluar, demi keamanan data. Fitur-fitur ini membuat website lebih fungsional, aman, dan user-friendly.
